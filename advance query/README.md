# TEST BASIC CRUD

### USAGE

- clone the project :

``` bash
git clone https://github.com/Stoic2002/test-farmagitechs.git
```

- open the folder :

``` bash
cd test-farmagitechs
```

``` bash
cd 'advance query'
```

- install package :

``` bash
npm install
```

- set up the .env file :

```
NODE_ENV=development
PORT=3000
DB_HOST=''
DB_USER=''
DB_PASSWORD=''
DB_NAME=''
MYSQL_PORT=''
```

- run server :

``` bash
npm run dev
```
